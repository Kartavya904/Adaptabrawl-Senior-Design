## Who this guide is for

This documentation is written for **players** who want to:

- Install and run Adaptabrawl on Windows or macOS
- Play local 1v1 or online 1v1 with a friend
- Understand controls, menus, and in-game features
- Adjust settings for comfort and performance

Technical terms are explained where they first appear, and we keep jargon to a minimum so you can get playing quickly.
